Introduction to Git

Git is an open source version control system that is used to track changes made to our code

Advantages of using Git

  • Speed and Performance
  • Distributed Version control
  • Collaboration
  • Conflict resolution
  • History and tracability
  • Branching and merging

Configuring Git

git config --global user.name "Adenike Adeyemi"

git config --global user.email "nikkystarry@gmail.com"

Basic Git commands

  1. git init: This is used to initialize a git repository
  2. git status: This is used to check the status of our file
  3. git add . : This is used to track the changes made to our file
  4. git commit -m "the commit message" : This is use to save the snapshot of our file using a descriptive caption
  5. git remote add origin: This is used to add a local repository to a remote repository
  6. git push origin main: This is used to push
